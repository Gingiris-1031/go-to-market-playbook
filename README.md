# 🧭 Go-to-Market Strategy Template

A fill-in GTM template that works for both B2B and B2C launches. Four worksheets you complete in order: **positioning canvas** (category, differentiator, value prop) → **ICP worksheet** (identify and rank ideal customer profiles) → **messaging framework** (headlines, elevator pitch, objection handling) → **channel-market fit** (match distribution channels to your buyer journey).

## Quick summary

- 🇺🇸 Not a lecture on GTM theory — a structured template you fill in for your own launch.
- 🇨🇳 不是 GTM 理论课，而是一套按顺序填完就能用的模板：定位画布 → ICP 工作表 → 信息框架 → 渠道匹配，B2B 和 B2C 通用。
- 🇯🇵 GTM理論の講義ではなく、順番に埋めるだけで使える構造化テンプレート。ポジショニング→ICP→メッセージング→チャネル適合。
- 🇰🇷 GTM 이론 강의가 아니라 순서대로 채우면 되는 구조화된 템플릿: 포지셔닝 캔버스 → ICP 워크시트 → 메시징 프레임워크 → 채널 적합성.
